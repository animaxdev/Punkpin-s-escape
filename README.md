# Punkpin-s-escape
Simple 2D game i've been writing for a couple weeks from scratch in C++ and SDL2.0. It's in early development. 

https://www.youtube.com/watch?v=f3frKCMhPyk

# Dependencies
```libsdl2-2.0-0
libsdl2-image-2.0-0
libboost-dev 
libsqlite3-dev
```

# How to run
Assuming all dependencies are installed:
```make
cd ./build && ./pumpkin
```
